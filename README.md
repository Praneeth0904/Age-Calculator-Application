## 📌 Age Calculator Web Application

Calculates exact age based on the user's date of birth.


---

## 🛠️ Tech Stack

HTML5 • CSS3 • JavaScript

---

## ✨ Features

* Calculates exact age based on date of birth.
* Displays results in years, months, and days.
* Handles date logic and edge cases effectively.
